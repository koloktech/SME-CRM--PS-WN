# Wan Boutique Order Manager

This is a vanilla HTML, CSS and JavaScript PWA for a personal-shopping business. It supports manual orders, WhatsApp-message parsing, manually entered item cost/service fees, payments, customer history, printing invoices, and Google Sheets through Apps Script.

## Local preview

Open the folder through a static server, then visit `index.html`. Without an Apps Script URL the app runs in safe browser-only demo mode.

## Deployment

Read the step-by-step guide in the project handoff message. The Apps Script database is generated by `setupDatabase()`; do not manually create or rename the data tabs.
